# Simple image compressor

A simple tool to compress Images and make them smaller for webdev.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install PIL.

```bash
pip install pillow
```

## Usage

```python
python main.py
```
## Settings

```python
quality = 0-100
output_extension = "jpg" or "jpeg" or "png"
```
